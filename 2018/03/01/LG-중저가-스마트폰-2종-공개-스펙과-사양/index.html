LG 중저가 스마트폰 2종 공개 스펙과 사양

LG가 모바일 월드 콩그레스에서 선보이는

K10과 K8 이라는 뛰어난 가성비를

자랑하는 실속형 스마트폰 2종을 발표하였습니다.

K8과 K10는 중저가 미드레인지 K 시리즈의 2 가지

새로운 엘지 스마트폰 입니다.

엘지는 스마트폰에 프리미엄 기능이 탑재되있다고

중장하고 있지만,

일부 카메라 기능을 제외하면 이 스마트폰이

애플과 삼성의 하이엔드 모델과

경쟁할 정도로 근접하진 않습니다.

예전 모델과 비교하면 디자인은 앞면의 유리와

뒤쪽 금속 프레임의 플라스틱으로

많이 변경되진 않았습니다.

두 전화기 색은 오로라블랙과 모로칸 블루, 테라 골드 3가지로 제공합니다.

이 2가지 기종의 특징은 카메라 기능에서 일반 미드레인지에서 보기 어려운 여러가지

기능을 탑재하여 강화하였습니다.

전면에 800만 화소 셀카를 장착한 K10은 선명한 화질과

셀피 촬영 했을때에 얼굴과 주위 배경을

분리하는 아웃포커싱 기능을 가지고 있습니다.

또한 자동초점 기능도 탑재하여 전작에

비하여 초점을 빠르게 잡습니다.

이 외에 손가락을 지문센스로 가져다 대는 것만으로

찰영 , 캡쳐 기능을 수행 하도록 한 '핑거터치' 기능과

후면에 홈 버튼으로 셀피를 찍고 보고있는 화면을

캡쳐하는 것도 가능합니다.

K10 사양

- 디스플레이 : 5.3인지 HD(1280 X 720 / 277ppi)

- 칩셋 : 1.5GHz Octa-Core

- 메모리 : 2gb 혹은 3gb 램 / 16gb 혹은 microSD (최대 2TB) / 32gb rom

- 배터리 : 3000 mAH

- 카메라 : 전면 8MP, 5MP(와이드) / 후면 13MP

- 크기 : 148.7 X 75.3 X 8.68mm

- 운영체제 : Android 7.1.2 Nougat

- 연결성 : Wi-Fi (802.11 b,g,n) 블루투스 4.2 / USB 2.0

- 네트워크 : LTE / 3G/ 2G

- 색상 : 오로라 블랙 / 모로칸 블루 / 테라 골드

LG K8은 어두운 환경의 촬영,

저고도 촬영모드가 더 업그레이드 되었으며,

셀피 촬영 때에는 밝은 흰색으로 변하는 화면이

플래시가 켜지는 것 같은 효과를 제공합니다.

K8 사양

- 디스플레이 : 5.0 인치 HD ON-CELL 터치 (1280 X 720 , 294ppi)

- 칩셋 : 1.3GHz 쿼드 코어

- 메모리 2gb 램 / 16gb rom / microSD (최대 32GB)

- 카메라 : 후면 8MP / 탈착가능한 전면 5MP배터리 : 2500 mAh

- 크기 : 146.3 x 73.2 x 8.2mm

- 운영체제 : Android 7.1.2 Nougat

- 네트워크 : LTE : 3G / 2G

- 연결성 : WI-FI (802.11 b, g, n) / 블루투스 4.2 / usb 2.0 유형 b

- 색상 : 오로라 블랙 / 모로칸 블루 / 테라 골드

- 기타 : 플래시 점프샷 / 플래시 / FM 라디오 / 퀵 셔터 / 빠른 캡쳐 / 시간 도우미

이번 LG의 미드레인지 신제품은 앞에서 말했든 카메라 기능을

강화했는데 연속 촬영된 20장의 사진을 GIF로 바꾸는 플래시 점프샷 기능과

찍은 사진을 즉시 SNS에 공유하는 퀵세어 기능, 화면상에서 초점을 잡고 싶은 곳에

화면을 터치하면 자동으로 촬영되는 심블뷰 기능 등 사용하기에 편한 다양한 카메라 기능이

업그레이드 됬습니다.
