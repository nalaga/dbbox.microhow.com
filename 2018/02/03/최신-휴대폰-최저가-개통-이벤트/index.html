최신 휴대폰 최저가 개통 이벤트

최신 휴대폰 최저가 개통 이벤트

자세히 보기 :https://goo.gl/SzH982

[최신 휴대폰 교체 이벤트]

- 아이폰8-64 · 아이폰8-256 · 아이폰8P-64 · 아이폰8P-256

- 갤럭시노트8 · 갤럭시노트8 256G

- S8 · S8 플러스 · S8 플러스-128

* 통신사 SK · KT · LG U+ 통신사 모두 개통 가능합니다.

* 미개봉 완박스 택배 발송 수령 후 개통 됩니다. 서울 · 경기 지역 퀵으로 당일 발송 가능합니다.

[혜택 1]

- 개통 고객 모든분께 12종 사은품 증정(개통 후 14일 이내 순차 발송)

(보조배터리 · 투얼 케이블 · 차량용 충전기 · 미니 셀카봉 · OTG 젠더 · 이어폰 줄감개 · 차량용 거치대 등)

- 번호 이동 고객 특별 혜택 : 정품 케이스+보호 필름+무선 충전기 지급

[혜택 2]

2018년 1월 개통 고객 대상 이벤트 자동 참여

- 1등 맥북(1명)

- 2등 스마트워치(5명)

- 3등 에어팟(10명)

- 4등 커피 상품권(50명) : 커피 상품권은 해당 시점 제휴 커피로 진행됩니다(스타벅스 · 이디야 · 투썸 등)

* 이벤트 추첨 번호는 개통하신 휴대폰 번호 4자리 입니다.

* 카카오톡 '플러스모바일' 플친 맺고 이벤트 관련 공지 소식으로 받아보세요.

[혜택 3]

- 개통 후 SNS에 개통 인증 포스팅 작성 하신 고객 전원에게 커피 상품권을 증정해 드립니다.

- 인증 방법 : SNS 포스팅 화면을 캡쳐하셔서, 플러스 카톡에 메시지를 남겨주세요.

* 정품 · 미개봉 폰만 취급합니다.

* 현금 완납 · 신용불량 개통 불

가합니다.
