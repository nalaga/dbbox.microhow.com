스마트폰용 블루투스 게임패드 7종 비교분석 (업데이트!)

NES30을 추가해 7종이 되었습니다. 더 추가되는 대로 또 업데이트 예정..

안드로이드용 블루투스 패드 7종 집중 비교 분석...은 페이크고 간단한 사용기 되겠습니다.

따로 적는 것 보다 하나에 적으면 읽는 분이나 쓰는 저나 도움이 되겠다 싶어서...

도 아니고 사용기 써야지 써야지 하다가 밀리고 쌓이고 하다보니 이렇게 되었습니다.

음... 이것은...음

최근에 몇가지 모바일 기기(안드로이드)용 블루투스 게임패드를 구입해 사용해 보았습니다. 물론 기존에도 몇몇 제품을 사용해 보았고요. 근데... 나 혼자 게임하는데 패드가 너무 많더군요.

제정신이 아닌 것 같아요. 이 많은 패드를 왜 계속 사들인 거지??

이제 정리가 필요한 것 같아요.

재미 없는 글이니 글 최하단의 요약 정리표만 보세요. 읽으면 데이터 낭비.

1. 조이트론 EX M Air

일단은 -중국oem이지만- 국내 정식 출시제품으로 정상적인 AS 등이 가능한 제품이라는 점이 다른 제품과 제일 큰 차별점이라 할 수 있습니다.

전용 클립을 통해 스마트폰을 패드에 고정 할 수 있으며 안드로이드, PS3, PC(유선) 연결이 가능합니다.

기본적으로 플레이스테이션 패드를 모방한 형태이고 그립감도 거의 유사합니다. 버튼 감도는 부드러운편이고 정확하고 빠르게 입력가능한 D-PAD부분이 장점입니다.

블루투스 2.0을 지원하며 안드로이드에서 아날로그패드를 마우스처럼 사용 가능하게 하는 기능 또한 이 제품의 장점.

클립이 우수하다

다만 충전 케이블이 마이크로USB가 아인 미니USB인 점, PC연결은 유선으로 지원된다는 점, 마우스 모드 시 움직임은 오른쪽 스틱/버튼은 왼쪽 범퍼라는 기괴한 배열이란 점 등이 단점으로 지적할 수 있습니다.

사실 다른 건 무시해도 되는데 결정적으로 입력에 딜레이가 발생하는 문제가 있습니다! 뭔가 밀렸다가 후루룩 입력되는 느낌이랄까. 커멘드를 많이 넣으면 발생하는 것 같은데 못쓸 정도는 아니고 그런데 민감한 분이 아니면 괜찮을 것 같기도하고...블투2.0이라 그런가?

(개인적으로 PS패드 배열인점도 단점!)

총평

기기마감 : ★★★★☆ (우수한 편이나 미끄럼 방지 고무가 마이너스. 고급지지못하다)

조작감도 : ★★★☆☆ (손에 느껴지는 감도는 우수하나 입력에 딜레이가 있다.엥?)

사용편의 : ★★★☆☆ (페어링, 모드변경 등 복잡하다.)

부가기능 : ★★★★★ (스마트폰용 클립우수, 마우스기능,접대용 PS3패드로 사용가능 등)

총점 : ★★★☆☆ (나쁘지 않은데 좀 아쉽다)

2. 매드캐츠 Micro C.T.R.L.r

콘솔 주변기기 제조사로 알려진 Mad Catz에서 발매한 안드로이드용 패드입니다. (아이폰용은 C.T.R.L.i 라는 이름으로 별도로 나와있습니다.)

패드 본체에 수납이 가능한 전용 스마트폰 고정 클립을 통해 스마트폰을 거치해 사용가능하며 블루투스 3.0을 지원합니다.

마이크로 라는 이름에 맞게 상당히 작은 사이즈(일반 게임 패드의 80% 사이즈)가 특징이며 내장 배터리가 아닌 AAA전지 2개를 사용하며 약 40시간 정도 사용 가능하다고 합니다!

하단의 선택버튼을 통해 (안드로이드)게임패드 모드, 마우스 모드, PC 모드로 간단히 변경 가능하고 블루투스를 통해 PC나 Mac용 컨트롤러로 사용가능합니다.

기타 미디어 컨트롤 기능이 있어 패드를 통해 음악 컨트롤도 가능.

아담한 느낌 / AAA건전지

D-PAD를 제외하면 그립감도 괜찮고 입력도 제법 잘되네요.

크기도 작아서 휴대/보관하기 편하고 여러모로 신경을 많이 쓴 패드이기는 하나 D-PAD부분은 사용이 거의 불가능할 정도의 감도를 가지고 있습니다. 게다가 유광 플라스틱의 외관이 스크레치에 매우 취약하여 금새 외관에 손상이 생기며 마감이 많이 부족한 스마트폰 클립등 2%부족한 완성도가 아쉬운 제품입니다. 고급스럽지 못해요.

총평

기기마감 : ★★★☆☆ (뭔가 부족한 느낌이든다)

조작감도 : ★★★★☆ (딜레이도 없고 다 괜찮은데 D-PAD가 망)

사용편의 : ★★★★★ (AAA건전지로 구동!, 부가기능이 많지만 상당히 직관적이다)

부가기능 : ★★★★☆ (마우스 기능,PC연결 기능,없는 것 보다 약간 나은 스마트폰 클립)

총점 : ★★★☆☆ (너도 나쁘지 않은데 좀 아쉽다)

3. 샤오미 게임패드 Xiaomi GameController

카피의 대명사인 샤오미 이기도하지만 이거 어찌 표현해야할지…

싫어할 수가 없는 샤오미입니다. Xbox one패드와 Ps4패드를 절묘하게 조합해 놓았습니다. 그것도 아주 잘.

기기 마감은 정말 중국제품인가 싶을 정도로 깔끔하고 매우 훌륭한 그립감을 보여줍니다. XBOX ONE패드를 쥐고 있는 듯한 느낌. 트리거는 PS4의 그것과 닯았습니다. 여러모로 신을 많이 쓴게 보이는 패드입니다. 조작감도 좋고 트리거에 스로틀(아닐로그)와 버튼 을 동시에 매핑해놔서 아날로그 입력을 인식 못하는 게임에서도 사용가능하게 해놨습니다.(보통은 버튼만 매핑)

블루투스3.0에 AA전지 2개로 구동된다는 점도 큰 장점.

xboxone 패드와 비교 / AA건전지!

듀얼 진동모터, 6축 자이로센서, 가속도 센서.. 말도 안되는 스펙의 게임 패드인데…

설명으로는 해당 기능을 지원하는 게임에서 사용 가능 하다고 합니다.

그런데 지원하는 게임이 없습니다. (이건 뭐지??)

진수성찬을 차려놨는데 숫가락 젓가락이 없어서 물만 마셔야하니 이건 있는 것도 아니고 없는 것도 아니고…

총평

기기마감 : ★★★★★ (아... 훌륭하다)

조작감도 : ★★★★★ (딜레이 없다, 느낌 좋다)

사용편의 : ★★★★★ (AA건전지로 구동!, 기능이 없어서 단순하다.응?)

부가기능 : ★☆☆☆☆ (PC연결가능하다. 부가기능이 없다.모터랑 G센서 왜 달아놨냐)

총점 : ★★★★★ (뭔가 잉여스럽지만 훌륭하다)

4. 넥서스 플레이어 게임패드 Asus gamepad for Nexus Player

구글TV 말아먹고 다시 한 번 도전해 만든 안드로이드TV와 함께 출시된 넥서스 플레이어용 게임패드입니다.

D-PAD부분은 PS4의 것과 비슷하고 배터리 수납부는 360패드를 따라 만든듯 한 모양새이지만 무려 구글에서 판매하는 레퍼런스패드!

블루투스3.0을 지원하고 버튼 감도 좋고 반응 빠르고 일단은 구글 공식패드라는 것! 그립 감도 적당하고 AA 건전지를 사용하기 때문에 여러가지로 관리도 편합니다.

무거워서 2개 들은줄 알았는데...

문제점은...

레퍼런스 패드란 녀석이 버튼배열이 엉망이에요. 가운데 있는 세개의 버튼이 문제!

모두가 사용하는 정상적인 배열이었다면 홈이 가운데로 가고 좌 우에 셀렉트와 스타트가 있어야 하는데...왼쪽부터 백버튼, 메뉴버튼, 홈버튼입니다.

생각없이 가운데 버튼이 홈 버튼이려니 생각하고 누르면 낭패. 생각없이 오른쪽 버튼이 스타트 버튼이려니 하고 누르면 완전 낭패(게임이 사라지고 홈으로감. 알면서도 무의식중에 계속 누르게됨)

거기에...트리거버튼 마감이 엉성합니다. 그리고 패드가 너무 무겁습니다. 안에 무게추가 들어있지 않은가 생각됩니다. 어느 정도는 그립감 올리는데 도움이 되지만 이건 좀 오버스러운 느낌.

그리고 더럽게 비쌈 (공식 가격 5만9천원)

총평

기기마감 : ★★★★★ (그래도 레퍼런스니까.)

조작감도 : ★★★★☆ (트리거 병맛, 아날로그 감도도 약간 이상)

사용편의 : ★★☆☆☆ (홈버튼 함정이 있으니 조심)

부가기능 : ★☆☆☆☆ (PC연결가능하다, 다른 안드로이드 기기와도 호환)

총점 : ★★★☆☆ (그래도 공식 패드니 사서 후회는 안한다)

5.8bitdo SNES30

사실 여기는 번외에 속하는데 소위 말하는 복각패드입니다. 이건 북미판 슈퍼닌텐도 버전이고 일본판 슈퍼패미콤버전도 있습니다.(SFC30)

다른 제품들과 마찬가지로 블루투스로 연결되며 내장 충전베터리를 이용합니다.

생김새가 모 제품과 똑같이 생겼고 그립감도 똑같고...아 추억돋는다.

의외로 제품 마감이 매우 훌륭하고 패키지도 상당히 괜찮게 꾸며놨습니다. 샤오미패드와 8bitdo 제품을 보면 이젠 중국제품은 엉성하다는 생각은 접어야 할 것 같습니다.

기능 개선 펌웨어 업데이트를 제조사에서 꾸준히 올려주는것도 주목할 점.

아이폰 연결도 가능하며 icade모드를 지원하고 동봉된 mircroUSB케이블을 이용하면 PC에서도 사용가능합니다. 조이트론패드가 PS3연결이 가능했던 것 처럼 이 제품은 Wii가 연결가능합니다?! Wii로 나온 클래식 게임을 더 클래식한 느낌으로!

다만 예전 패드 그대로 옮겨놔서 버튼이 많이 부족합니다. 정말 슈퍼패미콤 버튼, 딱 그 만큼만 있어요. 게다가 대각선 입력이 잘 안됩니다. 승룡권을 써야하는데 주먹이 나가서 낭패…

총평

기기마감 : ★★★★★ (오...케이블도 이쁘다. 열쇠고리도 주고 핸드폰그립도 주고)

조작감도 : ★★★☆☆ (버튼수는 적어도 옛날 느낌나서 좋다.근데 승룡권이 마음대로 안돼)

사용편의 : ★★★★☆ (기능이 많아서 약간 복잡. 펌웨어 업그레이드 해준다)

부가기능 : ★★★★☆ (PC, 아이폰, Wii 연결가능하다. 우왕)

총점 : ★★★☆☆ (이쁘다??. 캐주얼한 복고 에뮬 게이머에게는 별다섯일 수도)

6.8bitdo NES30 Pro

이건 약간 개념이 다른 모델인데 복각패드 컨셉으로 만든 최신패드?입니다.

그러니까 닌텐도패드의 느낌으로 만든 게임 패드입니다.(카피 아닌듯한 느낌인 듯한 느낌)

이것도 내장베터리 이용하는 블루투스 패드이고 상당히 컴팩트합니다. 매드캐츠 마이크로보다 더 작고 아담한 사이즈.

마찬가지로 마감이 뛰어나고 SNES 복각패드와 달리 버튼도 L2,L3에 아날로그까지 있을 건 다 있습니다. 동작 LED가 패드 전체를 감싸고 있어서 미적인 측면에서도 신경쓴 느낌.

안드로이드 이외에 PC, 아이폰, Wii 연결 가능하며 특이한 점은 PC연결시 xbox360패드 처럼 인식되어 xinput을 지원하는 최신게임들에서 사용가능합니다.

다만 인체 공학 따위는 고려하지 않던 구시대의 유물을 따라한 결과 그립감이 상당히 어색합니다.

크기도 작아서 뭔가 꽉 잡고는 있는데 잡힌 것 같지 않은 요상한 그립감입니다.

총평

기기마감 : ★★★★★ (오...LED, 케이블도 이쁘다. 열쇠고리도 주고 핸드폰그립도 주고)

조작감도 : ★★★★☆ (있을 건 다있고 잘 되는데 내손이 너무 크다)

사용편의 : ★★★★☆ (기능이 많아서 약간 복잡. 펌웨어 업그레이드 해준다)

부가기능 : ★★★★☆ (PC, 아이폰, Wii 연결가능하다. PC에선 360패드로 우왕)

총점 : ★★★★☆ (작아도 있을 건 다있는데 그립감이 별로)

7. 8bitdo NES30

다시는 패드 안사기로 마음먹고는 결국 또 샀습니다.

앞에 소개한 SNES30과 마찬가지로 닌텐도 복각패드입니다. 오리지널하고 다소 다른 점이 있는데 아무래도 버튼2개는 너무 적어서 인지 LR트리거를 포함해 6개의 버튼을 사용할 수 있습니다(스타트 셀렉트 포함하면 총 8개)

역시 내장 베터리 이용하고 microUSB 충전 및 유선 연결 지원합니다. 손에 딱 들어오는 작은 사이즈.에 다른 8bitdo 제품과 마찬가지로 깔끔한 마감.

SNES30과 "크게" 다른 점이 있다면 이녀석은 승룡권이 잘 들어갑니다! SNES30보다는 이녀석을 추천!

이것 역시 안드로이드 이외에 PC, Wii 연결 가능합니다.

총평

기기마감 : ★★★★★ (휼륭하다.)

조작감도 : ★★★★★ (승룡권 가능!, 근데 아날로그 스틱이 없어)

사용편의 : ★★★★☆ (펌웨어 업그레이드 가능. 다양한 기능 사용을 위해선 좀 복잡한 메뉴얼 필요)
