LG 유플러스(U+) 고객센타 LG스마트폰에서 비밀번호 인증으로 간단...

# LG스마트폰에서 LG 유플러스(U+) 고객센타를

비밀번호 인증(숫자4자리)로 간단하게 들어가는 방법!!

예전에는 스마트폰에서 고객센타 한번 들어갈려면 아디/비번을 알아야되서 많이 복잡하고

그래서 에잇~ 귀찮아서 가끔씩 PC에서 보는거 빼곤 못봤잖아요?(아닌가? 저만 그런가? ㅋㅋ^^;;)

근데 제가 G Pro2를 사용하고 나서부터는 LG 유플러스 고객센타를 정말 간단하게

들어갈수 있더라구요..ㅎㅎㅎ

그 방법은 바로 LG스마트폰에서 비밀번호 인증 숫자 4자리만 입력 하면 바로 고객센타를 들어갈수

있다는것!~

어떻게 들어갈수 있는지 알려드릴께요~

우선 LG 스마트폰 보시면 LG유플러스(U+)고객센타 라는 앱 이 있답니다.

LG U+ 고객센타 앱 을 클릭 눌러주세요~

그럼 U+ 고객센타 들어가기 전에 비밀번호 인증 / 홈페이지 Ohe ID 인증 요렇게 2가지 방법으로

들어갈수 있는게 나오는데요..

비밀번호 인증으로 LG U+ 고객센타를 간편하게 들어가자고요~ㅋㅋ

자동 로그인 옆에 비밀번호 설정/변경 을 클릭하세요~

그럼 개인정보 수집/이용에 동의하는지에 대해 묻는창이 뜬답니다.

체크하고 확인 눌러주세요~

그럼 비밀번호 설정 안내 창이 뜨는데요~

여기서 가지고 계시는 LG 스마트폰 본인에 대한 정보를 입력해야된답니다.

이름/생년월일/ 전화번호 입력 하고 SMS 인증요청 하시고 본인확인 눌러주세요~

(본인 아니면 근데 어려워요~ 아님 핸드폰 명의 정보를 아셔야됨 ^^;;)

그럼 이렇게 인증번호 다섯자리가 문자로 날라온답니다~

그럼 그 번호를 입력!

LG G Pro2는 그냥 저절로 문자온 번호가 인증번호 확인안에 뜨더라구요..ㅎㅎ

그래서 전 외울필요가 없어요~ㅎㅎㅎ

바로 본인확인 눌러줍니다~

본인확인이 완료되면 이제 비밀번호 설정 4자리 입력 하는 창이 뜬답니다.

자주 사용하는 비밀번호 4자리를 입력하세요~~

ㅋㅋㅋ 진짜 간편하죠?

이제 비밀번호 설정한 4자리 숫자로 간단하게 LG U+ 고객센타를 들어가자고요~

방금 입력한 4자리 비번 눌르고 로그인!~

헉... 제가 데이터를 이렇게 많이 썻나요? 흑흑 ㅠㅠ

이번달은 좀 자제해야겠네요~

암튼 요렇게 LG U+ 고객센타로 들어가면 메인화면에 데이타/음성+영상/메시지 잔여량을

확인할수 있답니다~

요금조회/소액결제/청구납부/나의 가입정보/맴버십 포인트/요금제 부가서비스 변경/신청 등

다양하게 확인하고 변경할수가 있어요~

소액결제는 요금조회 확인은 가능한데 차단/허용 등은 PC로 인터넷 들어가서 해야되는거 같아요~

그건 다시 알려드릴께요~ㅎㅎ

저는 이번달 사용요금이 얼마나왔는지 확인해보았답니다~ㅎㅎ

헉... 진짜 많이 나왔네요~~

좀 줄여야겠어요~~ㅠㅠ

암튼 요렇게 LG스마트폰으로 간단하게 비밀번호 인증하고 LG U+ 고객센타 들어가는 방법~

이제 아셨죠? ㅋㅋㅋ

바로 LG스마트폰으로 데이터/영상+음성/메세지 잔여량 확인 할수 있는방법이랍니다~ㅎㅎ
