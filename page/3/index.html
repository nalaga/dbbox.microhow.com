스마트폰 속도 향상 프로젝트

스마트폰은 저장매체로 SSD(Solid State Drive)를 사용합니다. 하드디스크도 마찬가지이지만 SSD의 경우 용량이 가득찬 경우 속도가 급격히 저하됩니다. 그래서 폰을 사용하실 때 항상 용량을 꽉 채우지 않고 어느 정도 여유를 두시고 주기적으로 청소하는 것이 좋습니다. 오늘은 쓸 때없이 차지하는 파일들을 청소해서 용량을 늘리는 방법을 알아보도록 하겠습니다.
