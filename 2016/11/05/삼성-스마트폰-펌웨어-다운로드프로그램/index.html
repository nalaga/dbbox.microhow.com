삼성 스마트폰 펌웨어 다운로드프로그램

이제 삼성 스마트폰의 펌웨어를 찾으려고 검색엔진들을 돌아다닐 필요가 없습니다

xda포럼에 zxz0O0 이라는 유저가 만들었다는 프로그램인데요

이 프로그램을 사용하려면 .NET Framework 3.5 와

2008, 2010년의 MS재배포패키지가 설치되어있어야합니다.

XDA원문보기

다운로드링크

SamFirm분할압축_001

SamFirm분할압축_002

SamFirm분할압축_003

SamFirm분할압축_004

SamFirm분할압축_005

SamFirm분할압축_006

파일을 다운받고 압축을 풀어주시고 SamFirm.exe 를 실행시켜줍니다.

왼쪽위에 Firmware Info 부분에서

Model에는 다운받을 삼성 스마트폰의 모델명

Region에는 해당스마트폰의 통신사코드를 입력해줍니다

통신사코드 보기

SKT = SKC KT = KTC U+ = LUC 자급제용 = KOO

여기에있는것을 넣어도 안된다면 Sammobile에서 기종을 검색하여 해당되는 코드를 넣어주세요

Auto와 Manual의 체크박스는

Auto의 경우 최신버전의 파일을 불러옵니다

Manual의 경우 버전을 집어넣으면 해당파일을 받을 수 있지만

귀찮으니 패스하고 그냥 Auto로받으세요

만약에 업데이트나 복구용이아닌 초기화전용 펌웨어를 받고싶으시다면

Binary Nature을 체크해주시면됩니다

그러고 다운로드를 누르면 파일이 SamFirm프로그램이 있는폴더에 다운됩니다

하지만!! 여기서 끝나는것이아닙니다.

제작자가 무슨생각으로 나누어놨는지는모르겠지만

다운로드만 누르면 enc4확장자의 암호화파일이 받아집니다

그렇기에 암호를 해제하는작업이필요한데요

별건아니고 다운로드가끝나면 활성화되는 Decrypt버튼을 누르시면 됩니다.

(거참 귀찮게... 한번에되게해주지...)

자 이제 다시폴더를 열어보시면 zip확장자의 압축파일이있습니다 이 압축을 풀으면 나오는

HOME이 펌웨어구요

BL이 피트입니다

나머지는... 오딘을 전부다알고있는것이아니라서요... 알아내게되면 업데이트하겠습니다
